## Security & Testing

### Importance of Testing

* Testing catches approximately **98% of software bugs**
* Testing is the most effective bug detection mechanism

### Required Test Case Components

Each test case should include:

1. Unique Identifier
2. Description
3. Prerequisites
4. Components Affected
5. Author
6. Requirements Covered
7. Test Steps
8. Pass/Fail Verification Criteria

### Security Layers

* Network Security
* Information Security
* Programming/Application Security
* Physical Security

### CIA Triad

The three primary security goals are:

* **Confidentiality** – Protect data from unauthorized access
* **Integrity** – Ensure data remains accurate and unaltered
* **Availability** – Ensure systems and data remain accessible
