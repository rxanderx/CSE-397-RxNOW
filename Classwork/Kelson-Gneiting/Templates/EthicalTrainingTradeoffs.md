## Part 2: Tradeoffs – ACM Code of Ethics vs. BYU-Idaho Honor Code
* Read the BYU-Idaho Honor Code
    [BYU-Idaho Honor Code](https://www.byui.edu/student-honor-office/ces-honor-code)
* Analyze Ethical Tradeoffs
    * Compare and contrast at least three ethical principles from the ACM Code of Ethics and the BYU-Idaho Honor Code.
    * Discuss how these two ethical frameworks align or conflict in areas such as workplace behavior, software development, and handling personal data.
    * Provide a scenario where a conflict between these ethical codes might arise and how you would navigate it.
* Example Scenarios:
    * Work Environment:
        * Conflict: The BYU-Idaho Honor Code emphasizes a commitment to honesty and personal integrity, while the ACM Code of Ethics places strong emphasis on whistleblowing unethical behavior. What happens if you witness unethical behavior in a company where reporting it might risk your job?
        * Resolution: You might need to weigh loyalty to your employer versus professional responsibility in reporting unethical practices.
    * Software Development:
        * Conflict: The ACM Code stresses avoiding harm, while an employer might pressure you to launch software quickly without adequate testing, increasing the risk of security flaws.
        * Resolution: Following ethical software development practices while maintaining professional integrity.
    * Handling Personal Data:
        * Conflict: Your company might collect personal data for marketing, but the ACM Code of Ethics emphasizes informed consent, while the BYU-Idaho Honor Code stresses honesty and integrity.
        * Resolution: Advocating for transparency in data collection policies.
