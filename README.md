# CSE-397-RxNOW

For the Professional Career Project, our team is developing RxNOW, a mobile application to streamline user medication management and reminders.
This repo contains streamlined documentation for each teammate's individual work for the class, and traceability for our team's work on the project. 
This is a repository for the simple documentation needed for our professor-mentor to evaluate our work.

# Classwork Section
Unrelated to the project each student has class deliverables of working on Career Center Gold Certification, Job Searching, and Achieving the AWS Cloud Practitioner Cert. 
Each student has a folder, where they will keep a log of their work on these individual projects.

# Teamwork Section
The majority of our work is done in the RxNow repository.
We will maintain links to the RxNow private repo for development, our kanban board for project and role issuing, and communication with our stakeholders for project overview clarity.
